THis is my first paragraph, In our daily activities, we push, pull, lift, carry, or move objects. Whenever a force is applied to an object and the object moves, work is said to be done. The ability to do work is called energy, while power tells us how fast the work is done. In science, work is said to be done when a force applied to an object causes the object to move in the direction of the force. 👉 If there is no movement, no work is done, even if force is applied.
